# CivilInfoHub

CivilInfoHub is a web-based platform designed to connect general users with small businesses and charity organisations. The system allows users to discover organisations, view profiles, save favourites, leave ratings and reviews, and find volunteer opportunities.

## Main Features

- User registration and login
- Business and charity organisation profiles
- Search and filtering for organisations
- Ratings and reviews
- Saved organisations
- Personalised recommendation system
- Monthly trend score and performance reports
- Volunteer opportunity listing and volunteer allocation
- Admin-controlled engagement weights for trend scoring

## Technology Stack

- **Frontend:** Vue.js
- **Backend:** Flask
- **Database:** PostgreSQL
- **Algorithms:** Matrix factorisation recommendation system, trend score calculation, and volunteer matching
